# Alan Fernandes | Nutricionista Esportivo

Site profissional para Alan Fernandes, Nutricionista Esportivo, desenvolvido com Vite, React, TypeScript e Tailwind CSS.

## Visão Geral

O projeto apresenta uma landing page responsiva com estética dark, esportiva e premium, voltada para conversão de visitantes em contatos para acompanhamento nutricional.

Principais seções:

- Hero com posicionamento direto
- Diferenciais do acompanhamento
- Público-alvo
- Método de trabalho
- Planos interativos
- Prova social preparada para conteúdos reais
- Sobre Alan Fernandes
- Conteúdos educativos
- FAQ
- CTA final

## Tecnologias

- Vite
- React
- TypeScript
- Tailwind CSS
- Framer Motion
- Lucide React

## Como Rodar

Instale as dependências:

```bash
npm install
```

Rode o ambiente de desenvolvimento:

```bash
npm run dev
```

Gere a versão de produção:

```bash
npm run build
```

Verifique o lint:

```bash
npm run lint
```

## Ajustes Antes da Publicação

- Trocar o `canonical` e os metadados `og:url` para o domínio final do site.
- Usar uma imagem absoluta no `og:image` quando o domínio estiver publicado.
- Inserir WhatsApp oficial, caso exista.
- Adicionar depoimentos, imagens e resultados reais quando forem enviados pelo cliente.
